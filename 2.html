This study guide explains motion in simple words. Read carefully, try the interactive parts, and think about the examples before using the simulation.

1. What is Motion?

Simple definition — Motion is a change in an object's position compared to a reference point.

Key Idea: Motion means something changes where it is when compared to something else.
When do we say something is in motion?
  • When it changes position over time compared to a reference point.
  • Example: A jeepney passing a sign — the jeepney is moving relative to the sign.
  • Example: A student walking past a classroom door.
  • Example: A ball rolling along the floor.
Why do we need a reference point?

We measure motion by comparing where something is to a fixed place or object. This fixed place is the reference point. Without it, we cannot tell if the object moved.

House    Pole    YOU    Tree
[H] ---- [P] ---- [Y] ---- [T]
       ^ reference point
If YOU move right, your position changes relative to the Pole.
Remember: Motion is always compared to a reference point.
Real-Life Connection: You feel still inside a moving jeepney, but you are moving relative to the road.
  • Car: Its position changes against buildings and signs.
  • Walking: Your position changes relative to the classroom door.
  • Ball rolling: It changes position along the floor compared to one corner of the room.

2. Position and Reference Point

Position tells us where an object is. The reference point is the "place" we use to compare.

What is position?

Position is the location of an object. We can write it as "5 meters from the door" or "2 meters north of the tree".

Static vs. moving reference point (simple):
  • Static reference point: A fix place like a pole or house. This is easiest to use.
  • Moving reference point: Sometimes we use a moving object as reference (like a car). This is more complicated because both objects move.
This picture shows a pole (reference) that does not move, and a person who moves from one place to another. We say the person's position changed relative to the pole.

A clear reference point helps us measure position and motion easily. A moving reference point makes it hard to tell which object moved unless we compare carefully.

3. Distance vs Displacement

These two are different: distance is the path; displacement is the straight line from start to end with direction.

Distance
  • Total path traveled.
  • Always positive (never negative).
  • Does not include direction.
Displacement
  • Straight-line distance from start to finish.
  • Has direction (e.g., 10 m north).
  • Can be zero if start and end are same place.

If you walk around the school courtyard and come back to the start, the distance you walked is the whole path, but your displacement is zero because you ended where you began.

4. Speed

Speed tells how fast an object moves.

Formula: speed = distance ÷ time
Units we often use: m/s (meters per second) and km/h (kilometers per hour).
Real-life: a car travels much farther than a runner in the same time.
Speed comparison table
Walking~1.5 m/s
Running~3 m/s
Bicycle~6 m/s
Constant speed = same distance each second. Changing speed = different distances each second.

To convert m/s to km/h, multiply by 3.6. (Because 1 m/s = 3.6 km/h)

5. Velocity

Velocity is speed with a direction.

If we say "10 m/s", that's speed. If we say "10 m/s north", that is velocity. Velocity changes when either speed or direction changes.

Remember: Two objects can have the same speed but different velocities if they move in different directions.

6. Acceleration (Intro)

Acceleration is any change in motion: speed up, slow down, or change direction.

We will not use complex formulas here. For Grade 7, we just need to know when acceleration happens and what it looks like.
Acceleration can be seen when a tricycle speeds up from a stop or slows down at a red light.
Important: Acceleration means a change in motion. It does not mean only speeding up.

7. Distance–Time Graph (Basic)

A simple graph helps us see motion over time.

On a distance–time graph:
  • The horizontal axis (x) is time.
  • The vertical axis (y) is distance.
  • A straight, slanted line upward = constant speed.
  • A flat (horizontal) line = not moving (rest).
  • Steeper line = faster speed.
Distance (m)
  |
  |        /--------  fast
  |       /
  |  ----/         slow
  | /
  |/             stopped
  +----------------------- Time (s)
In the example: the top slanted line is fast (steep). The low slanted line is slow. The flat line shows no movement.

8. Real-Life Applications of Motion

Motion is everywhere. Here are examples you know.

  • Riding a bike: You move forward; speed changes when you pedal harder.
  • Jeepney moving: It can go at a steady speed on a straight road.
  • Ball rolling: Slows down because of friction with the floor.
  • Person walking and stopping: Velocity changes when direction or speed changes.
  • Car slowing at a stoplight: It has negative acceleration (slows down).
Misconception Alert: "If I don't feel moving inside a vehicle, I am not moving." — Not true. You are moving relative to the road and buildings. Use a reference point to tell.
  • Ask students to watch a jeepney and count how many signs it passes in 10 seconds — then estimate speed (distance ÷ time).
  • Use a student walking across the classroom as a reference example for displacement and distance.

Wrap-up

In simple terms: Motion is change in position compared to a reference point. We measure how fast with speed, and we add direction to get velocity. Acceleration is any change in motion.

Remember: Always pick a clear reference point when you test motion in the simulation.
